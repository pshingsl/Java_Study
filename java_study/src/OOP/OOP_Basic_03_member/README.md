# ✅ 9. 필드

> 작성 일시: **2026-03-02  오후 6:03**
> 
필드는 객체의 상태 데이터를 저장하는 역할을 말한다.


## ✔ 선언 방법

```java
타입 변수명;
```

## ✔ 특징

- 객체 생성 후 사용 가능 -> 필드값을 읽고 변경하는 것을 말한다.
- 자동으로 기본값 초기화 -> null, 0, false 등으로 초기화
- 내부에서는 변수처럼 사용
- 외부에서는 `.`(도트 연산자)로 접근

### ✔ 객체 내부에서 필드 사용

객체 내부(생성자, 메서드)에서는  
단순히 **필드명만으로** 읽고 변경할 수 있다.
## 예시
```java
class Car {

    String color; // 필드

    void printColor() {
        color = "Red";        // 필드 값 변경
        System.out.println(color); // 필드 값 읽기
    }
}
```
### ✔ 객체 외부에서 필드 사용

객체 외부에서는 반드시  
**도트(.) 연산자**를 사용해야 한다.

```java
class Car {
    String color;
}

public class Main {
    public static void main(String[] args) {

        Car car = new Car();   // 객체 생성

        car.color = "Blue";    // 필드 값 변경
        System.out.println(car.color); // 필드 값 읽기
    }
}
```

### ✔ 핵심 정리

| 구분 | 접근 방법 |
|------|-----------|
| 객체 내부 | 필드명으로 직접 접근 |
| 객체 외부 | 객체.필드명 형태로 접근 |



### ⭐ 한 줄 핵심 정리

> 필드는 **객체가 생성된 후에만 존재하며**,  
> 내부에서는 바로 사용하고 외부에서는 **도트 연산자로 접근**한다.

# ✅ 10. 생성자
생성자는 객체 생성 시 필드를 초기화하는 특수한 메서드.

## ✔ 특징

- 클래스 이름과 동일
- 반환 타입 없음
- 객체 생성 시 자동 실행



## ✔ 기본 생성자
모든 클래스는 생성자가 존재하며, 하나 이상을 가질 수 있다. 개발자가 생성자를 작성하지 않으면 컴파일러는 기본 생성자를 자동으로 생성한다.

## 생성자 선언

객체를 생성할 때 **다양한 초기값을 설정**하기 위해 생성자를 직접 선언할 수 있다.

즉, 객체를 만들면서 동시에 **필드 값을 초기화**할 때 사용한다.



### ✔ 기본 문법

```java
클래스명(매개변수, ...) {
    객체 초기화 코드
}

```java
클래스명() {}
```
```
new 클래스명(매개값, ...);
```
## 예시
```
class Car {
String color;
}

public class Main {
public static void main(String[] args) {

        Car car = new Car();   // 객체 생성

        car.color = "Blue";    // 필드 값 변경
        System.out.println(car.color); // 필드 값 읽기
    }
}
```
## ✔ 생성자 오버로딩

객체를 다양한 방식으로 초기화하기 위해  
생성자를 여러 개 선언하는 것을 **생성자 오버로딩**이라고 한다.



### ✔ 정의

> 같은 이름의 생성자를 여러 개 선언하되  
> **매개변수의 개수, 타입, 순서가 다르게 정의되는 것**



### ✔ 왜 사용하는가?

객체를 다양한 방식으로 초기화하기 위해 사용합니다.
- 이름만 받는 생성자
- 이름과 나이를 받는 생성자ㅣ
---

### ✔ 조건

- 생성자 이름은 클래스 이름과 동일
- 반환 타입이 없음
- 매개변수의 개수 또는 타입 또는 순서가 달라야 함

# ✅ 11. 메서드
- 메소드 선언은 객체의 동작을 실행 블록으로 정의하는 것
- 메소드 호출은 실행 블록을 실제로 실행하는 것.


## ✔ 기본 구조

```java
리턴타입 메서드명(매개변수) {
    실행 코드;
}
```
## ✔ 리턴 타입

리턴 타입은 메서드 실행 후 **호출한 곳으로 전달되는 결과값의 타입**이다.

- 결과값이 없으면 → `void`
- 결과값이 있으면 → 반드시 `return` 필요



### ✔ 예제

```java
int add(int a, int b) {
    return a + b;
}
```
