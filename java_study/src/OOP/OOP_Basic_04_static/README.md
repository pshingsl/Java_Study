# ✅ 12. 인스턴스 멤버

> 작성 일시: **2026-03-02  오후 6:23**
> 
>

객체에 속한 멤버

- 객체 생성 후 사용 가능
- Heap 영역에 존재

### ✔ 예제

```java
class Car {

    int speed;   // 인스턴스 필드

    void run() { // 인스턴스 메서드
        System.out.println("달린다");
    }
}
```


---

## ✔ this 키워드

현재 객체 **자기 자신을 참조하는 키워드**이다.

### ✔ 사용 목적

- 필드와 매개변수 이름이 같을 때 구분
- 현재 객체의 멤버 접근

### ✔ 예제

```java
class Student {

    String name;

    Student(String name) {
        this.name = name; // 현재 객체의 필드
    }
}
```

# ✅ 13. static 멤버


정적 멤버는 **클래스에 소속된 멤버**이다.

- 객체 없이 사용 가능
- 공통 데이터 저장
-
### ✔ 특징

- Method Area에 저장
- 모든 객체가 공유
- 클래스 로딩 시 생성됨

### ✔ 선언 방법

```java
static 타입 변수;
static 리턴타입 메서드() {}
```


### ✔ 사용 방법

```java
클래스명.정적필드
클래스명.정적메서드()
```

## ✔ 사용 예시

```java
Math.PI
```

# ✅ 14. final 키워드

final은 **값을 변경할 수 없도록 만드는 키워드**이다.


### ✔ 특징

- 한번 초기화하면 변경 불가
- 상수 선언 시 사용

```java
final int MAX = 100;
```
