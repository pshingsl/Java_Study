## ✔ 객체란?


# ✅ 6. 클래스 선언



## ✔ 기본 문법

```java
public class 클래스명 {
}
```



## ✔ 이름 규칙

- 첫 글자 대문자
- CamelCase 사용

예:

```
Student
CarEngine
BankAccount
```

---

# ✅ 7. 객체 생성

객체는 new 키워드로 생성한다.


## ✔ 기본 문법

```java
클래스명 변수명 = new 클래스명();
```



## ✔ 예시

```java
Car car = new Car();
```



## ✔ 객체 생성 과정

1️⃣ Heap 영역에 객체 생성  
2️⃣ 생성자 실행  
3️⃣ 주소 반환  
4️⃣ 변수에 저장

---

# ✅ 8. 클래스 구성 요소 (3가지)


## 1️⃣ 필드 (Field)

객체의 데이터를 저장하는 변수

```java
String name;
int age;
```


## 2️⃣ 생성자 (Constructor)

객체 생성 시 초기화 담당

```java
Student() {
}
```



## 3️⃣ 메서드 (Method)

객체의 동작을 정의

```java
void study() {
    System.out.println("공부한다");
}
```
